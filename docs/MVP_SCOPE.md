# MVP Scope

## Included In MVP

### Landing Homepage

- Main hero section
- Worker value proposition
- Company value proposition
- Usage flow guide
- Payment option guide
- Trust indicators
- FAQ
- Consultation request or signup CTA

### Worker Webapp

- Signup
- Login
- Job category selection
- Preferred region selection
- Payment option selection
- Job list
- Job detail
- Apply to job
- Application history
- Confirmed work status
- My page

### Company Webapp

- Company signup
- Login
- Site registration
- Job post registration
- Applicant list
- Attendance confirmation
- Attendance status
- Work completion processing
- Expected settlement information
- Site dashboard

### Admin Webapp

- Worker management
- Company management
- Site management
- Job post management
- Application management
- Attendance management
- Report and inquiry management
- Basic statistics

## Excluded From MVP

- Native app
- AI auto matching
- Complex payroll automation
- Electronic contracts
- Insurance integration
- Foreign worker management
- Multilingual support
- Nationwide expansion
- Complex authorization structure
- Excessive animation
- Full skilled team contract system
- Advanced statistical reports
- Payment system
- Map features
- Real-time features
- Notification features
- Chat consultation system
- Real-time location tracking
- Automatic tax invoice issuing
- Complex labor management features
